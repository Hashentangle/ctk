# ctk


###ctk 节点挖矿程序


支持linux,可在centos下运行, peer.conf 可以配置矿工账号和证书加密串

其中矿工账号用于接收挖矿收益、证书加密串用于链接至区块链网络

miner = 钱包地址

cert_id = 证书加密串
